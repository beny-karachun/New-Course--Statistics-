# STAT//ACE — Probability & Statistics 094481

An exam-first interactive course built from the supplied Technion past-exam archive.

## Open the course

Open `index.html` in any modern browser. No installation or account is required. An internet connection is needed to load MathJax from its CDN.

Progress, mastered lesson exercises, best drill score, and memory-deck reviews are saved locally in the browser. Use the **BP** button in the top-right corner to reset them.

## What is included

- 8 focused teaching modules with 40 embedded practice exercises
- 15-question mixed drill with worked feedback
- 28-card repeat-until-automatic memory deck
- 22-minute timed exam sprint
- searchable course content
- printable final cheat sheet
- MathJax rendering for inline and display mathematics
- responsive desktop and mobile layouts

## Exam evidence used

The content and prioritization were checked against supplied Moed A exams and solutions spanning Spring 2020, Spring 2021, Winter/Spring 2022, Winter/Spring 2023, and Winter/Spring 2024. The newest supplied format contains 20 multiple-choice questions over three hours and permits two double-sided A4 aid sheets.

The recurring high-yield blocks were probability/distributions, sampling and CLT, confidence intervals and hypothesis tests, two-sample inference/Excel output, simple regression/ANOVA, and chi-square goodness-of-fit. Descriptive statistics appears, but with lower recurrence and less procedural depth.
